How to Update Your Website

  1. Make your changes in the form above
  2. Click "Save Changes & Download" to download the updated config.json file
  3. Go to your GitHub repository
  4. Upload the new config.json file, replacing the old one
  5. Commit the changes to your repository
  6. GitHub Pages will automatically update your website with the new content
